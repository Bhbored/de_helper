# DE Helper

> **A Simple Data Entry Assistant for POS Management**

**DE Helper** Designed to streamline data entry tasks. Instead of repeatedly interacting with a physical POS system, this app allows users to conveniently input data via their smartphone. The collected data can then be exported as an organized Excel sheet, significantly improving workflow efficiency for repetitive data entry tasks.

## üöÄ Getting Started

To run DE Helper locally, ensure you have Flutter installed.

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/Bhbored/de_helper.git
    cd de_helper
    ```

2.  **Get dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```
